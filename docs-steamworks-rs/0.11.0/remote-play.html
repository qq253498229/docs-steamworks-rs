RemotePlay (远程畅玩)

提供 Steam 远程畅玩会话相关信息的函数。 Steam 远程畅玩会将您的游戏内容流式传输至另一台电脑，或 Steam 流式应用或硬件。

1. 获取会话列表

pub fn sessions(&self) -> Vec<RemotePlaySession<Manager>>

返回所有活跃的远程畅玩会话列表。

2. 根据会话ID获取会话

pub fn session(
    &self,
    session: RemotePlaySessionId
) -> RemotePlaySession<Manager>

根据会话 ID 获取远程畅玩会话。 该会话可能有效也可能无效，也可能处于活跃或非活跃状态。
